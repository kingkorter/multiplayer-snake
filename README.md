# Multiplayer Snake Game (Python)

## Overview
A Python-based multiplayer Snake game with two players.  
Demonstrates:
- Object-oriented programming (OOP)
- Real-time game programming using `pygame`
- Handling multiple inputs and collisions
- Game loop and score tracking

## Features
- Two-player gameplay mimicing a hunter, prey envrironment(The bigger snake has more advantage over the smaller snake)
- Player 1 uses arrow keys, Player 2 uses W/A/S/D
- Snakes grows when eating food
- Collision detection (self, opponent)
- Score display for each player
- Restart functionality

## Project Structure
- `snake.py` – Game logic and main script
- `s.jpeg` – Image for the game (background)
- `food.png` - Image for the game (food)
- `requirements.txt` – Python dependencies
- `.gitignore` – Files/folders to ignore
